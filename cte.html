DFHS CTE Program

PLTW Engineering

PLTW Introduction to Engineering

This introductory PLTW class uses inventor to teach introductory Computer Aided Design (CAD). Students learn teamwork, business ideals, and engineering-related skills by working on hands-on projects throughout the year.

PLTW Principles of Engineering

Principles of Engineering is a hands on class in which students use VEX and other items in order to teach students the difference between theoretical and actual engineering. This class is an excellent introduction for future engineers as it teaches the hands on as well as theoretical engineering techniques.

PLTW Civil Engineering and Architecture

This class explores many aspects of construction planning and architectural design in order to prepare these students for Civil Engineering in the future. It teaches modern styles for house arrangements and the different construction planning methods.

PLTW Engineering Design and Development

In this class, students identify and solve a problem of their choosing. They are required to implement modern engineering problem solving techniques. This class teaches students about the business aspects of engineering in addition to the engineering researching methods.

Computer Science

Computer Programming 1

This class explores the fundamentals of computer programming. It teaches the basics to java programming as well as computer hardware and software. Computer Programming 1 allows students to gain practical experiences in a high level programming language.

Computer Programming 2

This class delves deeper into programming in java as well as iOS. It has an increased rigor, complexity, challenges, and creativity. Computer Programming 2 allows students to research more advanced topics in computer programming and counts as a research credit.

AP Computer Science A

This class emphasizes sorting and searching in the java language in order to prepare for work with large databases. This class teaches computational problem solving. This class also teaches about the internal memory of the computer, efficiency, and debugging.
